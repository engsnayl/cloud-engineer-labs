Title: Pipeline Failing — GitHub Actions Debugging
Difficulty: ⭐⭐ (Intermediate)
Time: 15-20 minutes
Category: CI/CD / GitHub Actions
Skills: GitHub Actions, workflow YAML, jobs, steps, environment variables, secrets

## Scenario

The GitHub Actions CI pipeline is failing. The workflow file has several syntax and configuration issues preventing builds from completing.

> **INCIDENT-CICD-001**: CI pipeline red for 2 days. Multiple failures in the GitHub Actions workflow. PR merges blocked. Dev team can't deploy.

## How to Use This Lab

1. Read the CHALLENGE.md for context
2. Examine the pipeline/workflow files
3. Find and fix the bugs
4. Run validate.sh to check your fixes

**Requires:** Understanding of CI/CD concepts. Some labs can be tested with `act` (local GitHub Actions runner) or Docker.
